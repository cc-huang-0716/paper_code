This study aims to investigate the impact of linear and nonlinear feature representations on stock price forecasting using high-frequency data from the Taiwan stock market. In terms of research methodology, we adopt a “dual-track feature engineering” approach: PCA is used for dimensionality reduction to represent the assumption of linear market structure, while the latent space extracted by an LSTM autoencoder represents the assumption of nonlinear market structure.

To rigorously validate the differences between the two, this study inputs these two sets of features into five machine learning algorithms—OLS, Ridge, Lasso, XGBoost, and Random Forest—for forecasting. We then use the Wilcoxon Signed-Rank Test to compare the absolute prediction errors between the two approaches, thereby testing whether nonlinear features possess a significant advantage.

Furthermore, to objectively confirm the existence of a nonlinear market structure, this study extracts the prediction residuals of the baseline linear model (PCA) during the test period. First, an ARIMA model is applied to filter out the residual linear autocorrelation, and then a BDS test is performed on the ARIMA residuals. By combining the Wilcoxon forecast performance test with the BDS residual nonlinearity test, this study comprehensively analyzes the empirical applicability of linear and nonlinear assumptions in the market.
